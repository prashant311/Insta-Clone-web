An application where users can upload images of their neighbourhood's dirty spots. The app should also has the functionality to classify them.

The application works in the spirit of the Swaach Bharat abhyan. Users can upload images of neighbourhoods, the application categorizes them as dirty or clean. Relevant authorities can then be notified of the same.

Dependencies :
  1.django
  2.clarifai
  3.sendgrid
  4.imgurpython
